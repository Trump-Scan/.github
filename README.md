# 트럼프 스캔 (Trump Scan)

## 프로젝트 소개

트럼프 대통령의 발언을 실시간으로 수집하고, 한국 경제와 시장에 미치는 영향을 분석하여 일반 투자자들에게 신속하게 제공하는 서비스입니다.

### 왜 만들었나요?

한국 경제는 트럼프 대통령의 발언 하나하나에 큰 영향을 받습니다. 하지만 이러한 정보가 한국까지 전달되는 데는 시간이 걸리고, 언어 장벽으로 인해 일반 투자자들은 정보 격차에서 불리한 위치에 있습니다. 

이 서비스는:
- **실시간 정보 제공**: 발언 발생 후 몇 분 내 피드 생성
- **언어 장벽 제거**: 영어 원문을 한국어로 요약
- **맥락 제공**: 한국 시장에 미치는 영향을 분석하여 제공

---

## 주요 기능

### 1. 실시간 발언 수집
- 여러 채널(Truth Social, 뉴스, 공식 발표 등)에서 트럼프 발언 모니터링
- 중복 제거를 통한 정확한 정보 제공

### 2. 한국어 요약
- 영어 원문을 간결한 한국어로 요약
- 핵심 내용을 빠르게 파악 가능

### 3. 한국 경제 영향도 분석
- 해당 발언이 한국 경제와 시장에 미치는 영향 분석
- 투자 결정에 도움이 되는 인사이트 제공

### 4. 관련 산업 식별
- 영향을 받을 산업 분류 (반도체, 자동차, 철강, 에너지 등)
- 산업별 필터링으로 관심 분야만 선택 조회

### 5. 피드 형태 제공
- 시간순으로 정렬된 피드
- 원문 링크 포함으로 출처 확인 가능
- 실시간 업데이트

## 시스템 개요

### 아키텍처

서비스는 5개의 레이어로 구성되어 있습니다:

```
[데이터 채널]
    ↓
[1. 데이터 수집]
    ↓
[2. 분석]
    ↓
[3. 중복 제거]
    ↓
[4. 피드 생성]
    ↓
[5. API]
    ↓
[프론트엔드]
```

### 레이어 설명

**1. 데이터 수집 레이어**
- 여러 채널에서 트럼프 발언 실시간 수집
- 이미 수집한 데이터는 스킵하여 효율성 확보
- 원본 데이터 저장

**2. 분석 레이어**
- 발언 내용 한국어 요약
- 한국 경제/시장 영향도 분석
- 관련 산업 식별
- 중복 제거를 위한 짧은 요약 생성

**3. 중복 제거 레이어**
- 임베딩 기반 의미 유사도로 중복 필터링
- 여러 채널에서 올라온 동일/유사 내용 제거
- 먼저 들어온 발언을 원본으로 유지

**4. 피드 생성 레이어**
- 분석 결과를 피드 형태로 구조화
- 피드 데이터 저장

**5. API 레이어**
- 프론트엔드에 피드 데이터 제공
- 시간순 정렬, 필터링, 검색 기능
- 실시간 새 피드 알림

### 데이터 흐름

1. **수집**: 다양한 채널에서 트럼프 발언 수집
2. **분석**: LLM으로 요약 및 영향도 평가
3. **검증**: 임베딩 기반 중복 제거
4. **저장**: 피드 형태로 저장
5. **제공**: API를 통해 프론트엔드에 제공

### 핵심 설계 원칙

- **속도**: 빠른 피드 생성을 최우선 목표로 설계
- **정확성**: 검증된 채널만 사용하고 중복 제거로 신뢰성 확보
- **완전성**: 다중 채널 모니터링으로 중요 발언 누락 방지
